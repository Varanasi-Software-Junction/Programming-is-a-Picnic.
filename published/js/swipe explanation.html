Swipe in Any Direction

Code Explanation

1. Detecting Touch Start: When the user touches the screen, we capture the starting coordinates (startX, startY).

2. Detecting Touch End: When the user lifts their finger, we capture the ending coordinates (endX, endY).

3. Calculating Swipe Direction: By comparing the difference (diffX, diffY), we determine if the user swiped left, right, up, or down.

4. Showing Alerts: If the swipe is significant (>50px), an alert box appears with the direction.

5. Works on Touchscreens: This code runs on mobile devices and tablets.
