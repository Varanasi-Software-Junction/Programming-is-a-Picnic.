CSS Variables Guide

What Are CSS Variables?

CSS variables allow you to define values once and reuse them throughout your CSS, making it easier to manage and update styles across your project. They are declared with a double dash -- and are often defined within the :root selector for global usage.

:root {
  --main-color: #3498db;
  --padding: 1rem;
}

Using CSS Variables

After defining a variable, you can insert its value anywhere in your CSS using the var() function.

:root {
  --main-color: #3498db;
  --secondary-color: #2ecc71;
  --font-size: 16px;
}

body {
  color: var(--main-color);
  font-size: var(--font-size);
}

button {
  background-color: var(--secondary-color);
  padding: var(--font-size);
}

Using CSS Variables in Different Contexts

CSS variables can be scoped to specific elements for component-specific customizations.

.card {
  --card-bg: #ecf0f1;
  background-color: var(--card-bg);
}

.card .title {
  color: var(--main-color); /* Uses global variable */
}

Fallback Values

CSS variables support fallback values in case a variable is undefined or unavailable.

p {
  color: var(--text-color, black); /* Fallbacks to black if --text-color is not defined */
}

CSS Variables in Media Queries

CSS variables can be updated inside media queries, useful for responsive design.

:root {
  --font-size: 16px;
}

@media (min-width: 600px) {
  :root {
    --font-size: 18px;
  }
}

body {
  font-size: var(--font-size);
}

Updating CSS Variables with JavaScript

CSS variables can be dynamically updated with JavaScript, useful for themes or interactive designs.

document.documentElement.style.setProperty('--main-bg', 'black');
document.documentElement.style.setProperty('--text-color', 'white');
