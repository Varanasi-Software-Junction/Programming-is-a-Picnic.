Matplotlib Pie Chart Guide

Introduction

The pie() function in Matplotlib's pyplot module is used to create pie charts, which illustrate numerical proportions as slices of a circle.

Syntax

matplotlib.pyplot.pie(x, explode=None, labels=None, colors=None, autopct=None, pctdistance=0.6, 
shadow=False, startangle=0, radius=1.0, counterclock=True, wedgeprops=None, 
textprops=None, center=(0, 0), frame=False, normalize=True, data=None)

Parameters

  • x (array-like): Input data for the slices.
  • explode (array-like): Fraction to offset slices for emphasis.
  • labels (list): Labels for each slice.
  • colors (array-like): Colors of the slices.
  • autopct (str or callable): Format for percentage labels.
  • pctdistance (float): Distance of percentage labels from the center.
  • shadow (bool): Adds a shadow effect.
  • startangle (float): Rotates the start angle of the pie chart.
  • radius (float): Radius of the pie chart.
  • counterclock (bool): Direction of slice progression.
  • wedgeprops (dict): Properties of the slices.
  • textprops (dict): Properties of text elements.
  • center (tuple): Shifts the center of the pie chart.
  • frame (bool): Adds a frame around the pie chart.
  • normalize (bool): Normalizes input data to sum to 1.

Examples

Basic Pie Chart

import matplotlib.pyplot as plt

sizes = [15, 30, 45, 10]
labels = ['Apples', 'Bananas', 'Cherries', 'Dates']

plt.pie(sizes, labels=labels, autopct='%1.1f%%', startangle=90)
plt.title('Fruit Distribution')
plt.show()

Exploding a Slice

explode = [0.1, 0, 0, 0]
plt.pie(sizes, explode=explode, labels=labels, autopct='%1.1f%%', shadow=True, startangle=140)
plt.title('Exploded Pie Chart')
plt.show()

Custom Colors

colors = ['gold', 'lightcoral', 'lightskyblue', 'yellowgreen']
plt.pie(sizes, labels=labels, colors=colors, autopct='%1.1f%%', startangle=90)
plt.title('Custom Colors Pie Chart')
plt.show()

Custom Wedge and Text Properties

wedgeprops = {'edgecolor': 'black', 'linewidth': 2}
textprops = {'fontsize': 12, 'color': 'blue'}
plt.pie(sizes, labels=labels, autopct='%1.1f%%', wedgeprops=wedgeprops, textprops=textprops)
plt.title('Styled Pie Chart')
plt.show()
