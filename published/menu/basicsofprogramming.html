Programming Concepts Overview

A summarized guide to essential programming concepts and terms.

Basic Concepts

  • Algorithm: A step-by-step procedure for solving a problem or performing a task.
  • Program: A set of instructions that a computer can execute.
  • Code: The actual text written in a programming language to create a program.
  • Syntax: The set of rules that defines the structure of a programming language.
  • Debugging: The process of identifying and fixing errors in code.

Literals, Variables, and Constants

  • Literals: Fixed values directly written in the code. Examples:
    • String Literal: "Hello, World!"
    • Numeric Literal: 42, 3.14
    • Boolean Literal: true, false
  • Variables: Named memory locations used to store data that can change during program execution. Example:
    • let age = 25;
  • Constants: Named memory locations used to store data that cannot be changed after initialization. Example:
    • const PI = 3.14159;
  • Data Types: The kind of value a variable or constant can hold. Examples:
    • Primitive: int, float, char, boolean
    • Composite: array, object, string
  • Scope: The context in which a variable or constant is accessible. Examples:
    • Local: Accessible only within a function or block.
    • Global: Accessible throughout the program.

Programming Languages

  • Machine Language: Binary code directly executable by the processor.
  • Assembly Language: Low-level language using symbolic instructions.
  • High-Level Language: User-friendly languages closer to human languages, such as Python, Java, and C++.
  • Scripting Language: Languages designed for automating tasks, often interpreted, such as JavaScript, Python, and Ruby.
  • Markup Language: Used for structuring and presenting content, such as HTML and XML.
  • Domain-Specific Language (DSL): Designed for specific tasks, such as SQL for database queries or CSS for styling.
  • Object-Oriented Language: Languages based on the concept of objects, such as Java, Python, and C++.
  • Functional Programming Language: Focuses on functional constructs, such as Haskell and Scala.
  • Procedural Programming Language: Based on procedures or routines, such as C and Pascal.
