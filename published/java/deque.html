Lesson: Java Deque

1. Introduction

The Deque (Double-Ended Queue) interface in Java represents a queue where elements can be added or removed from both ends...

2. Creating a Deque

import java.util.*;

public class DequeExample {
    public static void main(String[] args) {
        Deque deque = new LinkedList<>();
        deque.addFirst("Alice");
        deque.addLast("Bob");
        System.out.println(deque);
    }
}

3. Common Methods

Below are commonly used methods for Deque:

  • addFirst(E e) - Inserts an element at the front
  • addLast(E e) - Inserts an element at the end
  • removeFirst() - Removes and returns the first element
  • removeLast() - Removes and returns the last element

4. Using Deque

import java.util.*;

public class DequeOperations {
    public static void main(String[] args) {
        Deque deque = new LinkedList<>();
        deque.addFirst(10);
        deque.addLast(20);
        deque.addFirst(5);
        System.out.println(deque.removeLast()); // 20
        System.out.println(deque.peekFirst()); // 5
    }
}
