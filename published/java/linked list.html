1. Introduction

The List interface in Java represents an ordered collection that allows duplicate elements...

2. Creating a List

import java.util.*;

public class ListExample {
    public static void main(String[] args) {
        List names = new ArrayList<>();
        names.add("Alice");
        names.add("Bob");
        System.out.println(names);
    }
}

3. Common Methods

Below are commonly used methods for List:

  • add(E e) - Adds an element
  • get(int index) - Retrieves an element
  • remove(Object o) - Removes an element

4. Sorting a List

import java.util.*;

public class SortingExample {
    public static void main(String[] args) {
        List numbers = Arrays.asList(40, 10, 30, 20, 50);
        Collections.sort(numbers);
        System.out.println(numbers);
    }
}
