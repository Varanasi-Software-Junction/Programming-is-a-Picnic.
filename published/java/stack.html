1. Introduction

The Stack class in Java represents a last-in-first-out (LIFO) data structure...

2. Creating a Stack

import java.util.*;

public class StackExample {
    public static void main(String[] args) {
        Stack stack = new Stack<>();
        stack.push("Alice");
        stack.push("Bob");
        System.out.println(stack);
    }
}

3. Common Methods

Below are commonly used methods for Stack:

  • push(E e) - Pushes an element onto the stack
  • pop() - Removes and returns the top element
  • peek() - Returns the top element without removing

4. Using Stack

import java.util.*;

public class StackOperations {
    public static void main(String[] args) {
        Stack stack = new Stack<>();
        stack.push(10);
        stack.push(20);
        stack.push(30);
        System.out.println(stack.pop()); // 30
        System.out.println(stack.peek()); // 20
    }
}
