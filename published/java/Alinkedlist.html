Linked Lists in Java

Introduction

A Linked List is a linear data structure where elements, called nodes, are stored in a non-contiguous manner. Each node contains data and a reference to the next node in the sequence.

Relevant Packages

In Java, the LinkedList class is part of the java.util package. To use it, you need to import:

import java.util.LinkedList;

Class Members and Their Uses

Method Description
add(E e) Adds an element to the end of the list.
addFirst(E e) Adds an element at the beginning of the list.
addLast(E e) Adds an element at the end of the list.
remove() Removes the first element of the list.
get(int index) Retrieves the element at a specific index.

Use Cases with Code Samples

1. Implementing a Queue

import java.util.LinkedList;
import java.util.Queue;

public class QueueExample {
    public static void main(String[] args) {
        Queue queue = new LinkedList<>();
        queue.add(10);
        queue.add(20);
        queue.add(30);
        System.out.println(queue.poll()); // Removes 10
    }
}

2. Implementing a Stack

import java.util.LinkedList;
import java.util.Stack;

public class StackExample {
    public static void main(String[] args) {
        LinkedList stack = new LinkedList<>();
        stack.push(10);
        stack.push(20);
        stack.push(30);
        System.out.println(stack.pop()); // Removes 30
    }
}

Multiple Choice Questions (MCQs)

  1. Which package contains the LinkedList class in Java?
    a) java.lang
    b) java.util
    c) java.io
    d) java.linked
    Answer: b) java.util
  2. Which method removes the first element in a LinkedList?
    a) removeFirst()
    b) deleteFirst()
    c) popFirst()
    d) remove()
    Answer: a) removeFirst()
