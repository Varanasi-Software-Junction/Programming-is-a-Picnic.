Lesson: Java Queue

1. Introduction

The Queue interface in Java represents a first-in-first-out (FIFO) data structure...

2. Creating a Queue

import java.util.*;

public class QueueExample {
    public static void main(String[] args) {
        Queue queue = new LinkedList<>();
        queue.add("Alice");
        queue.add("Bob");
        System.out.println(queue);
    }
}

3. Common Methods

Below are commonly used methods for Queue:

  • add(E e) - Inserts an element into the queue
  • remove() - Removes and returns the front element
  • peek() - Returns the front element without removing

4. Using Queue

import java.util.*;

public class QueueOperations {
    public static void main(String[] args) {
        Queue queue = new LinkedList<>();
        queue.add(10);
        queue.add(20);
        queue.add(30);
        System.out.println(queue.remove()); // 10
        System.out.println(queue.peek()); // 20
    }
}
