How to Convert a React.js App into an APK

This guide will take you through the process of converting a React.js app (created using create-react-app) into an Android APK using Capacitor and Android Studio.

Step 1: Create a React.js App

First, create your React.js app using Create React App.

npx create-react-app my-react-app
cd my-react-app

Run the app to ensure it's working:

npm start

This should start the app on http://localhost:3000.

Step 2: Install Capacitor in Your React Project

Install Capacitor core and CLI:

npm install @capacitor/core @capacitor/cli

Initialize Capacitor:

npx cap init

Enter the name and the unique identifier (e.g., com.myreactapp.mobile) for your app when prompted.

Step 3: Build Your React App

Build the production version of your React app:

npm run build

Step 4: Add Android Platform to Capacitor

Once your React app is built, add the Android platform to your Capacitor project:

npx cap add android

Step 5: Copy React Build Files to Capacitor

Use Capacitor to copy the static build files to the Android project:

npx cap copy

Step 6: Open the Project in Android Studio

Open the Android project in Android Studio to build the APK:

npx cap open android

Step 7: Build the APK in Android Studio

In Android Studio, go to Build > Build Bundle(s) / APK(s) > Build APK(s).

Once the APK is built, you will find it in the following location:

android/app/build/outputs/apk/debug/app-debug.apk

Step 8: Test the APK on a Device

You can test the APK by transferring it to your Android device or by using adb to install it:

adb install android/app/build/outputs/apk/debug/app-debug.apk

Step 9 (Optional): Build a Signed APK for Production

If you want to publish your app on the Google Play Store, you need to build a signed APK:

In Android Studio, go to Build > Generate Signed Bundle / APK, and follow the steps to generate a signed APK.

Summary

By following these steps, you can successfully convert your React.js web app into a mobile APK using Capacitor and Android Studio. Capacitor wraps your web app inside a WebView for mobile app distribution.
