WordPress Learning Syllabus

1. Basics of WordPress

  • Introduction to WordPress: Overview and applications.
  • Setting up a local environment and installing WordPress.
  • Content Management: Posts, Pages, Categories, Tags, and Media Library.
  • Settings, Configurations, Security, and Backup.

2. Working with Themes

  • Theme Basics: Installing and activating themes.
  • Customization: WordPress Customizer, widgets, menus, and template hierarchy.
  • Creating Your Own Theme: Structure, templates, and styles.
  • Advanced Development: Custom templates, features, and optimizations.

3. Plugins in WordPress

  • Plugin Basics: Installing, activating, and managing plugins.
  • Creating Your Own Plugin: Hooks, actions, filters, and shortcodes.
  • Advanced Development: Database interactions, AJAX, and localization.

4. API Calling and Creation

  • Understanding APIs: REST vs. GraphQL, WordPress REST API.
  • API Calling: Fetching weather data using public APIs.
  • Creating Custom APIs: Custom endpoints for the latest posts.

5. Project Work

  • Build a portfolio site using a custom theme.
  • Create a plugin for specific functionality.
  • Implement a live weather widget using an API.
