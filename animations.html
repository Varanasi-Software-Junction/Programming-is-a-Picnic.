Lesson 2: CSS Animations

CSS Animations allow you to animate properties using keyframes. Below are detailed explanations of animation properties.

Keyframe Animation Example

In the example below, the box moves horizontally and changes color over time.


@keyframes moveBox {
    0% { transform: translateX(0); }
    50% { transform: translateX(200px); background-color: coral; }
    100% { transform: translateX(0); background-color: lightgreen; }
}
.animated-box {
    animation-name: moveBox;
    animation-duration: 4s;
    animation-timing-function: ease-in-out;
    animation-iteration-count: infinite;
    animation-direction: alternate;
}

Key Animation Properties

1. animation-name

This property defines the name of the keyframes to be used for the animation. In the example above, the `moveBox` keyframes are applied to the `.animated-box` class.

2. animation-duration

This property specifies how long the animation takes to complete one cycle. In the example, `animation-duration: 4s;` means the animation lasts for 4 seconds before looping or completing.

3. animation-timing-function

This property defines the speed curve of the animation. Common values include:

  • ease: The default value, starts slow, speeds up, then slows down.
  • linear: The animation has a constant speed throughout.
  • ease-in: The animation starts slow and speeds up.
  • ease-out: The animation starts fast and slows down.
  • ease-in-out: Starts slow, speeds up, then slows down.
  • cubic-bezier: Allows custom timing with control points.

In the example, `animation-timing-function: ease-in-out;` causes the animation to speed up and slow down smoothly.

4. animation-delay

This property specifies a delay before the animation starts. For example, `animation-delay: 2s;` would start the animation after a 2-second delay.

5. animation-iteration-count

This property defines how many times the animation should repeat. It can be a number or `infinite` for an endless loop.


.animated-box {
    animation-iteration-count: 3; /* The animation will repeat 3 times */
}

6. animation-direction

This property specifies whether the animation should play forwards, backwards, or alternate. Common values include:

  • normal: The animation plays forwards (default).
  • reverse: The animation plays backwards.
  • alternate: The animation alternates between forwards and backwards on each cycle.
  • alternate-reverse: The animation plays backwards on the first cycle, then alternates.

.animated-box {
    animation-direction: alternate; /* The animation will alternate directions */
}

7. animation-fill-mode

This property specifies how a CSS animation applies styles to its target before and after execution. Common values include:

  • none: No styles are applied outside the animation time (default).
  • forwards: After the animation ends, the final keyframe's styles remain applied.
  • backwards: Before the animation starts, the styles of the first keyframe are applied during the delay period.
  • both: Both `forwards` and `backwards` effects are applied.

.animated-box {
    animation-fill-mode: forwards; /* After the animation ends, final styles persist */
}

8. animation-play-state

This property specifies whether the animation is running or paused. This can be useful for starting, stopping, or pausing animations dynamically.


.animated-box {
    animation-play-state: paused; /* The animation will be paused */
}

Loading Spinner Example


@keyframes spinner {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
}
.loader {
    width: 50px;
    height: 50px;
    border: 5px solid #ccc;
    border-top-color: #333;
    border-radius: 50%;
    animation: spinner 1s linear infinite;
}

Typewriter Animation Example


@keyframes typewriter {
    from { width: 0; }
    to { width: 100%; }
}
.typewriter {
    font-family: monospace;
    font-size: 24px;
    width: 0;
    white-space: nowrap;
    overflow: hidden;
    border-right: 2px solid;
    animation: typewriter 4s steps(30) forwards;
}
