React Native Project Folder Structure

HelloWorldApp/
├── android/ # Android-specific files (Java, XML)
│ └── ... # Gradle, manifest, and app configurations
├── ios/ # iOS-specific files (Swift/Objective-C, Xcode)
│ └── ... # Xcode project files and app configurations
├── node_modules/ # Dependencies installed by npm
│ └── ... # JavaScript libraries used by the project
├── app.json # Basic app configuration (name, display name)
├── babel.config.js # Babel configuration file for JavaScript transformations
├── index.js # Entry point of the application
├── metro.config.js # Metro bundler configuration for custom setups
├── package.json # Lists dependencies, scripts, and project metadata
├── package-lock.json # Locks dependency versions for npm
├── README.md # Project overview and documentation
└── src/ # Custom folder for organizing your code (optional)
├── App.js # Main application component
└── assets/ # Static assets like images, fonts, etc. (optional)

Walkthrough: Creating and Running a React Native "Hello World" App

This section guides you through setting up and running a simple React Native "Hello World" app on Windows.

Step 1: Install Node.js and npm

First, install Node.js and npm if you haven't already:

  • Download Node.js from https://nodejs.org/.
  • During installation, check the option to install npm alongside Node.js.

Step 2: Install React Native CLI

Open your terminal and install the React Native CLI globally:

npm install -g react-native-cli

Step 3: Initialize a New React Native Project

Create a new React Native project using the following command:

npx react-native init HelloWorldApp

This command will create a new directory called HelloWorldApp with the necessary files and folders.

Step 4: Start Metro Bundler

Navigate to your project directory and start the Metro bundler:

cd HelloWorldApp
npx react-native start

Step 5: Run the App on an Android Emulator or Device

In a new terminal window, make sure you're still inside the project folder and then run:

npx react-native run-android

This command will build and run the app on your Android emulator or connected device.

Step 6: Edit App.js for "Hello World" Message

Open src/App.js (or HelloWorldApp/App.js if src folder is not used) and replace its content with:

import React from 'react';
import { View, Text, StyleSheet } from 'react-native';

const App = () => {
  return (
    
      Hello, World!
    
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#282C34'
  },
  text: {
    fontSize: 24,
    color: '#61DAFB'
  }
});

export default App;

Step 7: Save and Refresh

Save the changes in App.js. Your "Hello, World!" message should now be displayed on the app screen. If the app does not refresh automatically, you may need to reload it manually on the emulator or device.
