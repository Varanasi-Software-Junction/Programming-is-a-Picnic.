Calling SWAPI using cURL in PHP

Step-by-Step Walkthrough

In this tutorial, we will walk through how to call the Star Wars API (SWAPI) using cURL in PHP. Each step includes a test to ensure correctness.

1. What is cURL?

cURL (Client URL) is a PHP library used to make HTTP requests. It allows you to interact with APIs by sending GET or POST requests to a server, and it supports various protocols, including HTTP, HTTPS, FTP, etc.

Hereâ€™s an example of a basic cURL request:

<?php
// Initialize cURL
$curl = curl_init();

// Set URL
curl_setopt($curl, CURLOPT_URL, "https://swapi.dev/api/people/1/");

// Set option to return the response
curl_setopt($curl, CURLOPT_RETURNTRANSFER, true);

// Execute and fetch the response
$response = curl_exec($curl);

// Close cURL session
curl_close($curl);
?>

2. Initializing cURL

The first step is to initialize a cURL session using curl_init(). This returns a handle that can be used for further operations.

<?php
$curl = curl_init();
if ($curl) {
    echo "cURL initialized successfully!";
} else {
    echo "Failed to initialize cURL!";
}
?>

3. Setting cURL Options

We need to specify the URL to make a request to. For example, let's call the SWAPI endpoint to get details about Luke Skywalker.

<?php
curl_setopt($curl, CURLOPT_URL, "https://swapi.dev/api/people/1/");
curl_setopt($curl, CURLOPT_RETURNTRANSFER, true);
?>

4. Executing the cURL Request

Now that we've set the options, we can execute the cURL session to get the response.

<?php
$response = curl_exec($curl);
?>

5. Closing the cURL Session

After we have received the response, we should always close the cURL session to free up resources.

<?php
curl_close($curl);
?>

6. Decoding the JSON Response

The SWAPI response is in JSON format, so we need to decode it into a PHP array for easier access.

<?php
$data = json_decode($response, true);
?>
