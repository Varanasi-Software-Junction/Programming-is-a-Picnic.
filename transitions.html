CSS Transitions, Animations, and Transforms

Lesson 1: CSS Transitions

CSS transitions allow you to change property values smoothly (over a given duration).

Commonly Used Transitions

  • Background Color: Change background color smoothly.
  • Width: Smoothly change the width of an element.
  • Opacity: Fade elements in and out.
  • Transform: Apply transformations like scaling and rotation.
  • Box Shadow: Smoothly transition shadows for depth effects.

Example of CSS Transitions


.box {
    width: 100px;
    height: 100px;
    background-color: cornflowerblue;
    transition: background-color 0.5s ease;
}

.box:hover {
    background-color: coral;
}

Lesson 2: CSS Animations

CSS animations allow you to create complex animations using keyframes.

Keyframe Example


@keyframes slide {
    0% { transform: translateX(0); }
    50% { transform: translateX(100px); }
    100% { transform: translateX(0); }
}

.animated-box {
    width: 100px;
    height: 100px;
    background-color: teal;
    animation: slide 2s infinite;
}

Understanding Easing Functions

Easing functions define the acceleration curve of transitions or animations. Common easing functions include:

  • ease: Default easing function. Starts slowly, speeds up, and then slows down.
  • ease-in: Starts slowly and speeds up.
  • ease-out: Starts quickly and then slows down.
  • ease-in-out: Starts slowly, speeds up, then slows down.
  • linear: Constant speed throughout the transition.

Bézier Curves

Bézier curves provide more control over the timing of animations:


.custom-ease {
    transition: transform 0.5s cubic-bezier(0.25, 0.1, 0.25, 1);
}

Customize the curve by adjusting the control points.

3D Transforms and Angle Transforms

CSS 3D transforms allow for manipulation of elements in three dimensions, adding depth and immersion to your designs.

3D Transform Functions

  • rotateX(deg): Rotates the element around the X-axis.
  • rotateY(deg): Rotates the element around the Y-axis.
  • rotateZ(deg): Rotates the element around the Z-axis.
  • translateZ(px): Moves the element closer or farther from the viewer.
  • perspective(px): Sets the perspective depth of the 3D scene.

Example: 3D Rotation


.rotate-box:hover {
    transform: rotateX(45deg) rotateY(45deg);
}

Example: Perspective Effect


.perspective-box:hover {
    transform: translateZ(100px);
}

Example: rotate3d() Function


.rotate3d-box:hover {
    transform: rotate3d(1, 1, 0, 180deg);
}

Angle-Based Transforms

Use angles to create various effects:

Rotate Example


.rotate-angle-box:hover {
    transform: rotate(60deg);
}

Skew Example


.skew-box:hover {
    transform: skewX(30deg) skewY(15deg);
}

Combined Transform Example


.combined-transform-box:hover {
    transform: rotateX(45deg) rotateY(45deg) translateZ(50px) skewX(10deg);
}
