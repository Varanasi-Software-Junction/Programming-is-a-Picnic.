Lesson: Using var, dynamic, nullable, and Future in Dart

Overview

In this lesson, we will explore how to use the following types in Dart functions:

  • var
  • dynamic
  • nullable
  • Future

1. Using var

var is a keyword in Dart that allows you to declare a variable without explicitly specifying its type. The type is inferred based on the assigned value.

Example 1: Using var

void main() {
  var name = 'Aarav'; // Inferred as String
  var age = 25;       // Inferred as int

  print('Name: $name, Age: $age');
}

2. Using dynamic

The dynamic type allows you to define variables that can hold values of any type. Unlike var, which is statically typed, dynamic can change types at runtime.

Example 2: Using dynamic

void main() {
  dynamic variable = 'Hello'; // Initially a String
  print(variable); // Output: Hello

  variable = 42;   // Now an int
  print(variable); // Output: 42

  variable = true; // Now a bool
  print(variable); // Output: true
}

3. Using Nullable Types

Dart's null safety feature allows you to explicitly define whether a variable can hold a null value. You use the ? symbol to indicate that a type can be null.

Example 3: Using Nullable Types

void main() {
  String? name; // Nullable String
  print(name); // Output: null

  name = 'Diya';
  print(name); // Output: Diya
}

4. Using Future

In Dart, Future is used for asynchronous programming, allowing you to work with values that may not be immediately available. Functions that return a Future are typically used for operations that take time, such as network requests.

Example 4: Using Future

Future fetchUserData() async {
  // Simulating a network delay
  await Future.delayed(Duration(seconds: 2));
  return 'User data fetched';
}

void main() async {
  print('Fetching user data...');
  String userData = await fetchUserData(); // Awaiting the Future
  print(userData); // Output: User data fetched
}

Putting It All Together: Example Code

Future getUserInfo(String? id) async {
  if (id == null) {
    return null; // Return null if id is null
  }
  
  // Simulating a network delay
  await Future.delayed(Duration(seconds: 1));
  return 'User info for ID: $id';
}

void main() async {
  var userId = '12345'; // Inferred as String

  dynamic userInfo = await getUserInfo(userId);
  print(userInfo); // Output: User info for ID: 12345

  String? anotherUserId; // Nullable String
  userInfo = await getUserInfo(anotherUserId);
  print(userInfo); // Output: null
}

Summary

  • var: Use var when you want to declare a variable with type inference.
  • dynamic: Use dynamic when you want a variable that can hold any type and may change at runtime.
  • Nullable Types: Indicate that a variable can hold a null value by using the ? symbol.
  • Future: Use Future for asynchronous operations, allowing you to wait for a result that may not be immediately available.
