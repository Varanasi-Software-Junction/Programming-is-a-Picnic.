Lesson: Defining and Calling Functions in Dart

Overview

This lesson builds upon the previous introduction to functions in Dart. We will cover the following additional concepts:

  • Required parameters
  • Recursive functions
  • Passing functions as parameters

What is a Function?

A function is a reusable block of code that performs a specific task. Functions can take inputs, perform operations, and return outputs.

Why Use Functions?

  • Code Reusability: Write once, use multiple times.
  • Modularity: Break down complex problems into smaller, manageable parts.
  • Readability: Functions can make code easier to read and understand.

Defining Functions

In Dart, a function is defined using the following syntax:

ReturnType functionName(ParameterType parameterName) {
  // Function body
}

Required Parameters

You can enforce that certain parameters must be provided when calling a function. This is done using the required keyword for named parameters or by defining positional parameters without default values.

void createProfile({required String name, required int age}) {
  print('Profile created: Name: $name, Age: $age');
}

Example: Using Required Parameters

void main() {
  createProfile(name: 'Aarav', age: 25); // Valid call
  // createProfile(name: 'Diya'); // This will cause an error since 'age' is required.
}

Recursive Functions

A recursive function is a function that calls itself to solve a smaller instance of the same problem. Recursion is often used for tasks like calculating factorials or navigating data structures.

Example: Calculating Factorial Using Recursion

int factorial(int n) {
  if (n <= 1) {
    return 1; // Base case: factorial of 0 or 1 is 1
  } else {
    return n * factorial(n - 1); // Recursive case
  }
}

Example: Calling Recursive Function

void main() {
  int number = 5;
  int result = factorial(number);
  print('The factorial of $number is: $result'); // Output: The factorial of 5 is: 120
}

Passing Functions as Parameters

Dart allows you to pass functions as parameters to other functions. This is particularly useful for callbacks or when implementing higher-order functions.

Example: Passing Functions as Parameters

void printResult(int value, Function operation) {
  int result = operation(value); // Calls the passed function with value
  print('The result is: $result');
}

int doubleValue(int number) {
  return number * 2; // A simple function to double the value
}

int squareValue(int number) {
  return number * number; // A simple function to square the value
}

Example: Using Passed Functions

void main() {
  printResult(5, doubleValue); // Output: The result is: 10
  printResult(4, squareValue);  // Output: The result is: 16
}

Best Practices

  • Use Descriptive Names: Function names should clearly indicate what the function does.
  • Keep Functions Short: Aim for functions that perform one task.
  • Document Your Functions: Use comments to describe the purpose, parameters, and return values.
  • Avoid Side Effects: Functions should avoid modifying variables outside their scope unless necessary.

Complete Example Code

// Required Parameters Example
void createProfile({required String name, required int age}) {
  print('Profile created: Name: $name, Age: $age');
}

// Recursive Function Example
int factorial(int n) {
  if (n <= 1) {
    return 1; // Base case
  } else {
    return n * factorial(n - 1); // Recursive case
  }
}

// Function Passing Example
void printResult(int value, Function operation) {
  int result = operation(value); // Calls the passed function with value
  print('The result is: $result');
}

int doubleValue(int number) {
  return number * 2; // A simple function to double the value
}

int squareValue(int number) {
  return number * number; // A simple function to square the value
}

void main() {
  // Calling Required Parameters
  createProfile(name: 'Aarav', age: 25);

  // Calling Recursive Function
  int number = 5;
  int result = factorial(number);
  print('The factorial of $number is: $result'); // Output: The factorial of 5 is: 120

  // Calling Functions with Passed Functions
  printResult(5, doubleValue); // Output: The result is: 10
  printResult(4, squareValue);  // Output: The result is: 16
}
