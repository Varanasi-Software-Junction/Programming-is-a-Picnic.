Introduction

Dart provides several control structures for decision-making and looping, such as if-else, loops, switch-case, and null-aware operators. These structures control the flow of your program based on conditions or repetitive needs.

If-Else Statement

The if-else statement in Dart allows you to execute code blocks based on conditions.

Basic Syntax

if (condition) {
    // Execute if true
} else {
    // Execute if false
}

Example

void main() {
    int age = 18;
    if (age >= 18) {
        print("You are an adult.");
    } else {
        print("You are a minor.");
    }
}

Flowchart

If-Else-If Ladder

if (condition1) {
    // Code if condition1 is true
} else if (condition2) {
    // Code if condition2 is true

    // Code if none of the conditions are true
}

Switch-Case Statement

The switch-case statement is used to choose between multiple options based on a single variable.

Syntax

switch (variable) {
    case value1:
        // Execute if variable == value1
        break;
    case value2:
        // Execute if variable == value2
        break;
    default:
        // Execute if none match
}

Null-Aware Operators

Dart provides the ?. (null-aware operator) and ?? (null-coalescing operator) to simplify code when working with potentially null values.

Null-Aware Operator ?.

String? name;
print(name?.length);  // Safe access to length, returns null if name is null

Null-Coalescing Operator ??

String? name;
String defaultName = name ?? "Guest";  // Use "Guest" if name is null
