HTML Tables Learning Page

1. Using <thead>, <tbody>, and <tfoot>

To improve accessibility and organization, you can use the <thead>, <tbody>, and <tfoot> tags:

  • <thead>: Contains header information for the table, such as column titles.
  • <tbody>: Contains the main content of the table, such as rows of data.
  • <tfoot>: Contains footer information, often used for totals or summary rows.

Using these tags improves the table's accessibility by clearly separating sections for screen readers, making it easier for users to navigate through the table content.

Example of <thead>, <tbody>, and <tfoot> Usage:

Player Runs Fours Sixes
Player 1 75 8 3
Player 2 45 4 1
Player 3 100 10 4
Total: 220/3 in 20 overs 
<table>
    <thead>
        <tr>
            <th>Player</th>
            <th>Runs</th>
            <th>Fours</th>
            <th>Sixes</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Player 1</td>
            <td>75</td>
            <td>8</td>
            <td>3</td>
        </tr>
        <tr>
            <td>Player 2</td>
            <td>45</td>
            <td>4</td>
            <td>1</td>
        </tr>
        <tr>
            <td>Player 3</td>
            <td>100</td>
            <td>10</td>
            <td>4</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="4">Total: 220/3 in 20 overs</td>
        </tr>
    </tfoot>
</table>

2. Table with colspan and rowspan Attributes

This example demonstrates colspan and rowspan to merge cells:

Item Category Details
Laptop Electronics Brand A Price: ₹50000
Electronics Brand B Price: ₹60000 
<table>
    <tr>
        <th>Item</th>
        <th>Category</th>
        <th colspan="2">Details</th>
    </tr>
    <tr>
        <td rowspan="2">Laptop</td>
        <td>Electronics</td>
        <td>Brand A</td>
        <td>Price: ₹50000</td>
    </tr>
    <tr>
        <td>Electronics</td>
        <td>Brand B</td>
        <td>Price: ₹60000</td>
    </tr>
</table>

4. Registration Form

This example uses a table structure to create a registration form:

Registration Form
First Name:
Last Name:
Email:
Password:
Country: 
<table>
    <tr><th colspan="2">Registration Form</th></tr>
    <tr><td>First Name:</td><td><input type="text" name="first_name" /></td></tr>
    <tr><td>Last Name:</td><td><input type="text" name="last_name" /></td></tr>
    <tr><td>Email:</td><td><input type="email" name="email" /></td></tr>
    <tr><td>Password:</td><td><input type="password" name="password" /></td></tr>
    <tr><td>Country:</td><td><select name="country"><option>India</option><option>USA</option><option>UK</option></select></td></tr>
    <tr><td colspan="2"><button type="submit">Register</button></td></tr>
</table>
