Understanding 2D Arrays in C

Introduction

A 2D array in C is essentially an array of arrays. It can be visualized as a matrix with rows and columns. Hereâ€™s a simple syntax for declaring a 2D array:

// Declaring a 2D array in C int matrix[3][3]; // A 3x3 matrix

Declaring 2D Arrays

1. Standard Declaration

You can declare a 2D array with specific dimensions:

int arr[3][4]; // A 2D array with 3 rows and 4 columns

2. Initialization at Declaration

You can also initialize a 2D array when declaring it:

int arr[2][3] = {{1, 2, 3}, {4, 5, 6}}; // Initialized 2D array

3. Using VLA (Variable Length Arrays)

C allows the use of variable length arrays:

int rows = 3, cols = 4;
int arr[rows][cols]; // VLA

4. Dynamic Memory Allocation

2D arrays can also be created using dynamic memory allocation:

int **arr;
arr = malloc(rows * sizeof(int*));
for (int i = 0; i < rows; i++) {
    arr[i] = malloc(cols * sizeof(int));
}

Matrix Operations

1. Matrix Addition

Matrix addition involves adding corresponding elements of two matrices of the same size.

#include <stdio.h> int main() { int A[2][2] = {{1, 2}, {3, 4}}; int B[2][2] = {{5, 6}, {7, 8}}; int C[2][2]; // Resultant matrix // Performing matrix addition for (int i = 0; i < 2; i++) { for (int j = 0; j < 2; j++) { C[i][j] = A[i][j] + B[i][j]; } } // Printing the result printf("Resultant Matrix C:\\n"); for (int i = 0; i < 2; i++) { for (int j = 0; j < 2; j++) { printf("%d ", C[i][j]); } printf("\\n"); } return 0; }

2. Matrix Multiplication

Matrix multiplication involves multiplying rows of the first matrix with columns of the second matrix. The number of columns in the first matrix should match the number of rows in the second matrix.

#include <stdio.h> int main() { int A[2][2] = {{1, 2}, {3, 4}}; int B[2][2] = {{5, 6}, {7, 8}}; int C[2][2] = {0}; // Resultant matrix // Performing matrix multiplication for (int i = 0; i < 2; i++) { for (int j = 0; j < 2; j++) { for (int k = 0; k < 2; k++) { C[i][j] += A[i][k] * B[k][j]; } } } // Printing the result printf("Resultant Matrix C:\\n"); for (int i = 0; i < 2; i++) { for (int j = 0; j < 2; j++) { printf("%d ", C[i][j]); } printf("\\n"); } return 0; }

Passing 2D Arrays to Functions

1. Passing a 2D Array

When passing a 2D array to a function, you must specify the number of columns:

#include <stdio.h> void printMatrix(int arr[2][3]) {
for (int i = 0; i < 2; i++) {
for (int j = 0; j < 3; j++) {
printf("%d ", arr[i][j]);
}
printf("\\n");
}
} int main() {
int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}};
printMatrix(matrix);
return 0;
}

2. Passing a 2D Array Using Pointers

You can also pass a 2D array as a pointer:

#include <stdio.h> void printMatrix(int (*arr)[3], int rows) {
for (int i = 0; i < rows; i++) {
for (int j = 0; j < 3; j++) {
printf("%d ", arr[i][j]);
}
printf("\\n");
}
} int main() {
int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}};
printMatrix(matrix, 2);
return 0;
}

Test Questions

Question 1: What will be the output of the following code snippet?

int A[3][3] = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};
printf("%d", A[1][2]);

Answer: The output will be 6, as A[1][2] refers to the element in the second row and third column, which is 6.

Question 2: How would you declare a 2D array with 5 rows and 4 columns?

Answer: int matrix[5][4];

Coding Assignments

Assignment 1: Write a program to calculate the transpose of a 3x3 matrix.

Solution:

#include <stdio.h> int main() {
int A[3][3] = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};
int transpose[3][3]; // Calculating transpose for (int i = 0; i < 3; i++) {
for (int j = 0; j < 3; j++) {
transpose[j][i] = A[i][j];
}
} // Printing the transpose printf("Transpose of Matrix A:\\n"); for (int i = 0; i < 3; i++) {
for (int j = 0; j < 3; j++) {
printf("%d ", transpose[i][j]);
}
printf("\\n");
} return 0;
}
