C Programming: Conditionals and Operators

1. if-else Statement

The if-else statement is used to make decisions in C. The code inside the if block is executed if the condition is true, otherwise, the code inside the else block is executed.

if (condition) {
    // code if condition is true
} else {
    // code if condition is false
}

Example:

#include <stdio.h>

int main() {
    int num = 10;
    if (num > 5) {
        printf("Number is greater than 5.\\n");
    } else {
        printf("Number is 5 or less.\\n");
    }
    return 0;
}

2. Ternary Operator

The ternary operator is a shorthand for if-else. It evaluates a condition and returns one of two values based on whether the condition is true or false.

condition ? expression_if_true : expression_if_false;

Example:

int num = 10;
int result = (num > 5) ? 100 : 200;
printf("%d\\n", result); // Output: 100

3. Logical Operators

Logical operators are used to combine multiple conditions in C.

  • && (Logical AND): True if both conditions are true.
  • || (Logical OR): True if at least one condition is true.
  • ! (Logical NOT): Inverts the truth value.

Examples:

Logical AND:

int a = 5, b = 10;
if (a > 0 && b > 0) {
    printf("Both numbers are positive.\\n");
}

Logical OR:

int a = 5, b = -10;
if (a > 0 || b > 0) {
    printf("At least one number is positive.\\n");
}

Logical NOT:

int a = 0;
if (!a) {
    printf("a is zero.\\n");
}

4. Operator Precedence and Associativity

Operator precedence determines the order in which operators are evaluated in expressions. Associativity determines the direction in which operators of the same precedence are evaluated.

Operator Precedence Table:

Operator Symbol Description Associativity
Parentheses () Grouping expressions Left to Right
Logical NOT ! Logical negation Right to Left
Multiplication, Division *, / Multiplication, division Left to Right
Addition, Subtraction +, - Addition, subtraction Left to Right
Relational <, >, <=, >= Less than, greater than Left to Right
Equality ==, != Equal to, not equal to Left to Right
Logical AND && Logical AND Left to Right
Logical OR || Logical OR Left to Right
Ternary ?: Ternary operator Right to Left
Assignment =, +=, -= Assignment operators Right to Left

Example with Precedence and Associativity:

int a = 5, b = 10, c = 0;
if (a > 0 && b > 0 || c == 0) {
    printf("Condition is true.\\n");
}

5. switch Statement

The switch statement evaluates a variable and matches it against different cases. If a match is found, the corresponding code is executed.

switch (variable) {
    case value1:
        // code for case 1
        break;
    case value2:
        // code for case 2
        break;
    default:
        // code if no cases match
}

Example:

#include <stdio.h>

int main() {
    int day = 2;
    switch (day) {
        case 1:
            printf("Monday\\n");
            break;
        case 2:
            printf("Tuesday\\n");
            break;
        case 3:
            printf("Wednesday\\n");
            break;
        default:
            printf("Other day\\n");
    }
    return 0;
}
